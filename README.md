## Project Description
TaskManager is a powerful and flexible application designed to help individuals and teams manage their tasks efficiently. It aims to simplify task management through an intuitive interface, allowing users to focus on what matters most.

## Features
- **User Authentication:** Secure login/signup functionality.
- **Task Management:** Create, edit, and delete tasks with ease.
- **Due Dates:** Set due dates to keep tasks organized.
- **Prioritization:** Assign priority levels to tasks for better organization.
- **Collaboration:** Share tasks with team members and collaborate in real time.
- **Notifications:** Get reminders for upcoming deadlines.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js, Python
- **Database:** SQL lite

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Armpa16/TaskManager.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TaskManager
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage Guide
- Register a new account or log in to your existing account.
- Start creating tasks by clicking on the 'New Task' button.
- Fill in the task details and save it to your task list.
- Use filters to view tasks by priority or due date.
- Collaborate with teammates by sharing tasks and assigning them to others.
